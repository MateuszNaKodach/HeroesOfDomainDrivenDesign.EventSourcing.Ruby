# README

## How to run?

Install ruby on your machine and Rails:
`gem install rails`

0. `bundle install`
1. `docker compose up`
2. `rails db:create`
3. `rails db:migrate`
4. `rails server`

Fixing format errors:
`rubocop --auto-correct`
